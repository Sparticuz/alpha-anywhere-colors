Installation
============

### Installation

1.  Move the `*.a5t` files to your `%APPDATA%\Alpha Software\Alpha Anywhere Version 12` directory.
2.  In Alpha Anywhere, in the Control Panel, open Edit->Settings. Navigate to System Colors. Click Load.
3.  Select the preferred a5t file.

Atom Color scheme for Alpha Anywhere
=============================

# Atom Dark UI theme

A dark UI theme for Atom.

This theme is installed by default with Atom and can be activated by going to
the _Themes_ section in the Settings view (`cmd-,`) and selecting it from the
_UI Themes_ drop-down menu.
